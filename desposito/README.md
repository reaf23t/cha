# Nirserve - Sistema de Repositório Organizado

Sistema simplificado de armazenamento de arquivos com foco em organização automática por categorias.

## 🚀 Funcionalidade Principal

- **Upload por Pastas**: Ao enviar um arquivo, você seleciona a categoria (Arduino ou Lego) e o sistema o salva automaticamente na pasta correspondente.
- **Armazenamento Persistente**: Os arquivos salvos permanecem no sistema mesmo após atualizar a página.
- **Interface Limpa**: Foco total na tarefa de upload e visualização organizada.

## 📂 Pastas Designadas

1. **Arduino**: Destinada a programações (.ino, .cpp) e modelos 3D (.stl, .obj).
2. **Lego**: Destinada a projetos EV3, Spike Prime e scripts Python relacionados.

## 🛠️ Como Usar

1. Selecione os arquivos no campo de upload.
2. Escolha a pasta designada (Arduino ou Lego).
3. Clique em **Salvar**.
4. O arquivo aparecerá instantaneamente na pasta correta.
